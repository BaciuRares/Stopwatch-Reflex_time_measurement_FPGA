# Stopwatch-Reflex_time_measurement_FPGA
VHDL-based system for measuring human reaction time, developed in Vivado for Artix-7 FPGAs. The architecture features a modular RTL design, including a clock frequency divider, a multiplexed 7-segment display driver, and a dedicated control unit. Covers the full design flow from VHDL coding to XDC constraints.
